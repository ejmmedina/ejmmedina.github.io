Web Application for Basic Image Pre-processing Techniques

By: Elijah Justin Medina

Introduction

Currently, neural networks have widespread applications on image processing from preprocessing, feature extraction, segmentation, and object recognition. The ability of such models to perform these processing techniques are unparalleled, especially in comparison with traditional techniques. What is lost, however, is the process as to how these images are transformed from one point to the next, that is, the interpretability of the models. Although not as powerful, traditional techniques use intuitive methodologies, instead. That is not to say that these methods are have no use anymore. In fact, these methods can supplement the built algorithms to improve their performance and interpretability. In this project, we explore some of the most basic image processing techniques: thresholding and the fundamental morphological operations.

Process

In the video, the first step after selecting an image is to create a mask which will separate the foreground and the background. This is done by selecting a minimum and maximum threshold value for the red, green, and blue channels of the image. All the pixels within this range of thresholds are considered as the foreground while those that are outside are the background.

However, since most images do not always have a clearly defined separation of background and foreground from the color values, this process usually yields a rough segmentation of the foreground and background. A useful technique to remedy this is the group of techniques, morphological operations. There are many operations but all of which can be reduced to a combination of two: dilation and erosion. By iterating a structuring element (a matrix of some shape) through each foreground pixel, the pixels are expanded or eroded by the structuring element. In the aforementioned description, the foreground pixels are the 1s in the mask and the background pixels are the 0s. A more detailed explanation is available here.

Admittedly, this project is insufficient in properly processing an image as it only performs one operation, either a dilation or an erosion. By combining these operations can one only properly perform image processing.
